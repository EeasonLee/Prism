## 上下文

## 决策

- 有哪些备选方案？

## 使用说明

1. 复制此模板创建新的 ADR
2. 文件名格式：`adr-XXX-简短标题.md`
3. 编号从 001 开始递增
4. 更新 `docs/architecture/README.md` 中的索引

## ADR 状态说明

- **提议** - 正在讨论中，尚未决定
- **已接受** - 已决定采用，正在实施
- **已拒绝** - 已决定不采用
- **已废弃** - 曾经采用，但已被替代
- **已替代** - 被新的 ADR 替代
